Liten bot for å monitorere sensur fra FSWEB.

## Utvikling

### Generelt
- [ ] Logge inn på FSWEB
- [ ] Aksessere sensur
- [ ] Persistent tilgang
- [ ] Monitorere sensur

### Varsling
- [ ] Discord
- [ ] Telegram

## Installering
Prosjektet er basert på TypeScript, så nødvendige pakker må bli installert. Prosjektet forutsetter at du har Node 16.0.0 eller nyere installert.

```
npm install
```

## Run and gun
```
npm start
```
